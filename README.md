# Aditia Nugroho Portfolio

Elegant single-page React portfolio for Aditia Nugroho, built with React, Tailwind CSS, Framer Motion animations, Font Awesome icons, and Jest unit tests.

## Structure

- `public/index.html` - React application shell.
- `src/App.jsx` - Main app and theme state.
- `src/components/` - Header, About, Experience, Skills, Projects, Education, Contact Form, and Dark Mode Toggle.
- `src/styles/tailwind.css` - Tailwind source CSS.
- `src/tests/` - Jest and React Testing Library tests.
- `assets/images/aditia-nugroho-casual.jpg` - Portfolio profile image.

## Run Locally

```bash
npm install
npm start
```

Visit `http://localhost:3000`.

## Run Unit Tests

```bash
npm test
```

The test suite covers component rendering, contact form submission, theme switching, and skill score calculation.
